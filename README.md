# dacs.run

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### deploy on github and gitlab
```
chmod +x deploy.sh
./deploy.sh
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
